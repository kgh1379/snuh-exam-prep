## 1. 감사규정 핵심 → [[감사규정, 부정청탁금지법, 이해충돌방지법 Quiz]]

| 항목 | 핵심 내용 |
|------|----------|
| 감사의 직무 | 재산상황 감사, 회계감사, 원장 의뢰 업무 (기재부장관 지시 ✕) |
| 감사인 자격 | 공인회계사, 회계/감사 2년↑, 필요 지식/경험자 (민간기업 경력 ✕) |
| 일상감사 대상 | 1백만원↑ 경비, 50만원↑ 업무추진비/회의비, 2천만원↑ 공사, 3백만원↑ 전도금 |
| 일상감사 제외 | 인건비, 국내출장/부임여비, 세금공과금/공공요금 |
| 감사보고 기한 | 감사종료 후 20일 이내 보고서 작성, 2개월 이내 통보 |
| 청탁금지법 선물 | 일반 5만원, 농수산물 15만원 (명절 전후: 30만원) |
| 경조사비 | 축의·조의금 5만원 + 화환 합산 10만원 이내 |
| 외부강의 상한 | 병원소속: 시간당 40만원, 총액 60만원 (국제기구: 지급자 기준) |

## 2. 설치법/정관/직제 핵심 → [[설치법, 병원정관, 직제규정 Quiz]]

| 항목 | 핵심 내용 |
|------|----------|
| 설치법 목적 | 고등교육법에 따른 의학교육·연구·진료 |
| 이사회 구성 | 이사장(서울대 총장) 포함 이사 9명 + 감사 1명 |
| 당연직 이사 | 서울대총장, 교육부차관, 복지부차관, 기획예산처차관, 병원장, 의대학장, 치과병원장 |
| 임명직 이사 | 2명 |
| 이사회 의사정족수 | 일반: 재적과반출석+출석과반찬성, 원장추천 등: 재적2/3이상 찬성 |
| 정기이사회 | 2월, 12월 |
| 원장 임기 | 3년, 한 번 연임 가능 |
| 원장 임명 | 이사회 추천 → 교육부장관 제청 → 대통령 임명 |
| 직무대행 순서 | 진료부원장 → 소아진료부원장 → 공공부원장 → 행정처장 |
| 병원관리위원회 | 위원장: 병원장, 정기회: 분기1회, 의결: 재적2/3출석+출석2/3찬성 |
| 전결권자 범위 | 과장 및 팀장 이상 보직자 |

| 징계사유 시효 | 발생일로부터 3년 (금품수수/횡령: 5년) |
| 정년퇴직 전 연수 | 1년 이내 |
| 명예퇴직 | 20년↑ 근속, 정년퇴직일 1년↑ 전에 신청 |
